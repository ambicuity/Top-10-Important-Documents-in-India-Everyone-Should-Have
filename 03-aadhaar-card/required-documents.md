# Required Documents for Aadhaar Card

## Overview

For Aadhaar enrollment, you need to provide:
- **Proof of Identity (POI)** - Any ONE document
- **Proof of Address (POA)** - Any ONE document
- **Date of Birth proof (DOB)** - Any ONE document (optional but recommended)

**Note**: Some documents can serve multiple purposes (e.g., Passport serves as POI, POA, and DOB proof)

## Documents for Different Categories

### For Adults (18+ Years)

**Required**:
- One POI document
- One POA document
- One DOB proof (optional but helps prevent issues)

**Best Combination**:
- Voter ID (POI + POA)
- Birth Certificate (DOB)

OR

- Passport (POI + POA + DOB) - Single document covers all!

### For Children (Below 5 Years) - Baal Aadhaar

**Documents Required**:
- **Child's Birth Certificate** (Mandatory)
- **Parent's Aadhaar Card** (Mandatory)
- One parent must be present during enrollment

**Documents NOT Required**:
- No separate POI or POA for child
- Child's photographs will be taken at center

**Note**: Only photograph captured (no fingerprints or iris scan)

### For Children (5-18 Years)

**Documents Required**:
- Birth Certificate (DOB proof)
- School ID Card with photograph (POI)
- Parent's Aadhaar Card (POA - same address assumed)

OR

- Birth Certificate
- Parent's Aadhaar Card (for POI and POA)
- School Leaving Certificate (additional support)

**Note**: Parent/guardian must accompany child

### For Homeless/Destitute Persons

If you don't have any documents:

**Introducer System**:
- Find an introducer who has Aadhaar
- Introducer must be resident of same locality
- Introducer's biometrics will be captured
- Introducer's Aadhaar card required
- "Declaration by Introducer" form to be filled

**Documents from Introducer**:
- Introducer's Aadhaar card
- Introducer's address proof
- Declaration form with photograph

### For NRIs (Non-Resident Indians)

**Required Documents**:
- Passport (mandatory - serves as POI, POA, and DOB)
- Indian address proof (any document from POA list)
- Proof of stay in India for 182+ days

**Additional**:
- Visa with entry stamps
- Return ticket details

## Special Cases and Alternatives

### If You Don't Have Aadhaar and Need Documents

**Chicken-and-Egg Problem**: Many documents need Aadhaar, but Aadhaar needs documents!

**Solution**:
1. Use **Passport** (if you have) - accepted everywhere
2. Use **Voter ID** (easy to get without Aadhaar)
3. Use **Driving License** (can apply without Aadhaar in some states)
4. Use **Ration Card** (state-specific)
5. Use **Birth Certificate** + School certificates

### If Address on Documents Doesn't Match

**Option 1**: Update documents to current address first
**Option 2**: Provide multiple documents showing address change trail
**Option 3**: Use parent's/spouse's address with relationship proof

### If Living in Rented Accommodation

**Documents Accepted**:
- Rent agreement (notarized)
- Letter from landlord with their address proof
- Electricity/water bill in your name
- Bank statement with current address

### If Staying with Parents/Relatives

**Documents Accepted**:
- Parent's/relative's Aadhaar card
- Relationship proof (birth certificate for parents)
- Rent-free accommodation letter from parent/relative
- Parent's/relative's address proof

## Document Format Requirements

### Original Documents
- Must bring originals for verification
- Originals will be returned immediately
- Photocopies will be taken by operator

### Document Validity
- Bills should not be older than 3 months
- Property tax receipt valid for 1 year
- Other documents should be currently valid (not expired)

### Document Quality
- Clear and legible
- All pages intact
- No tampering or alterations
- Color copies preferred for documents with photos

## Documents NOT Accepted

❌ Self-attested photocopies (originals must be shown)
❌ Expired documents
❌ Bills older than specified period
❌ Handwritten receipts without official stamp
❌ Tampered or altered documents
❌ Documents with incorrect information

## Important Points

✅ **One document can serve multiple purposes** (e.g., Passport for POI, POA, DOB)
✅ **Name should match across documents** (small variations acceptable)
✅ **Address format variations acceptable** (e.g., "Street" vs "St.")
✅ **Keep originals safe** - they will be returned immediately
✅ **No need to carry all documents** - just one each for POI and POA
✅ **If unsure, carry multiple options** - operator will select best one

## Common Document Combinations

### Combination 1 (Easiest)
**Passport alone** - Covers POI, POA, and DOB

### Combination 2 (Most Common)
**Voter ID** - POI and POA
**Birth Certificate** - DOB

### Combination 3 (Alternative)
**PAN Card** - POI and DOB
**Electricity Bill** - POA

### Combination 4 (For Students)
**School ID Card** - POI
**Parent's Aadhaar** - POA
**Birth Certificate** - DOB

### Combination 5 (For Working Professionals)
**Company ID Card** - POI
**Bank Statement** - POA
**PAN Card** - DOB

## Document Checklist Before Visiting Center

Before going to Aadhaar enrollment center, ensure:

- [ ] Original documents (not photocopies)
- [ ] Documents are currently valid (not expired)
- [ ] Bills are recent (within 3 months)
- [ ] Name is consistent across documents
- [ ] Address is clear and complete
- [ ] Photographs on documents are clear
- [ ] All pages of document present

## FAQs About Documents

**Q: Can I use digital/e-documents?**
A: No, physical original documents required for verification.

**Q: What if my name is slightly different across documents?**
A: Minor variations are acceptable (e.g., "Kumar" vs "Kumr"). Major differences may need correction.

**Q: Can I use someone else's address?**
A: Yes, with proper relationship proof or letter from address owner.

**Q: What if I recently shifted to a new address?**
A: Use any valid POA document with new address, even if only one document shows it.

**Q: Do I need to carry photocopies?**
A: No, center will make photocopies. Just bring originals.
