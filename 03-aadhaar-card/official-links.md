# Official Links and Resources for Aadhaar Card

## Main UIDAI Portal

### Official Website
- **URL**: [https://uidai.gov.in](https://uidai.gov.in)
- **Purpose**: Main portal for all Aadhaar-related information and services

### Resident Portal
- **URL**: [https://myaadhaar.uidai.gov.in](https://myaadhaar.uidai.gov.in)
- **Services**: 
  - Update Aadhaar details
  - Download e-Aadhaar
  - Check Aadhaar status
  - Lock/unlock biometrics
  - Generate VID

## Important Service Links

### 1. Enroll for Aadhaar

**Find Enrollment Center**
- **URL**: [https://appointments.uidai.gov.in/easearch.aspx](https://appointments.uidai.gov.in/easearch.aspx)
- **Purpose**: Locate nearest Aadhaar enrollment center

**Book Appointment**
- **URL**: [https://appointments.uidai.gov.in](https://appointments.uidai.gov.in)
- **Purpose**: Schedule appointment for Aadhaar enrollment

### 2. Download e-Aadhaar

**Download e-Aadhaar**
- **URL**: [https://eaadhaar.uidai.gov.in](https://eaadhaar.uidai.gov.in)
- **Options**: Download using Aadhaar number, Enrollment ID, or Virtual ID

**Get Aadhaar on Mobile (m-Aadhaar)**
- **Android**: [Google Play Store - m-Aadhaar](https://play.google.com/store/apps/details?id=in.gov.uidai.mAadhaarPlus)
- **iOS**: [App Store - m-Aadhaar](https://apps.apple.com/in/app/maadhaar/id1435469474)

### 3. Check Aadhaar Status

**Verify Aadhaar Generation Status**
- **URL**: [https://myaadhaar.uidai.gov.in/check-aadhaar-status](https://myaadhaar.uidai.gov.in/check-aadhaar-status)
- **Require**: 14-digit Enrollment ID

**Verify Aadhaar Number**
- **URL**: [https://myaadhaar.uidai.gov.in/verifyAadhaar](https://myaadhaar.uidai.gov.in/verifyAadhaar)
- **Purpose**: Check if your Aadhaar is valid and active

### 4. Update Aadhaar Details

**Online Update Portal**
- **URL**: [https://ssup.uidai.gov.in/ssup](https://ssup.uidai.gov.in/ssup)
- **Services**: Update name, address, date of birth, gender, mobile, email

**Update Aadhaar at Center**
- **URL**: [https://appointments.uidai.gov.in/easearch.aspx](https://appointments.uidai.gov.in/easearch.aspx)
- **Purpose**: Find centers for physical update

**Document Update Portal**
- **URL**: [https://myaadhaar.uidai.gov.in/update-aadhaar](https://myaadhaar.uidai.gov.in/update-aadhaar)

### 5. Order Aadhaar PVC Card

**Order PVC Card**
- **URL**: [https://myaadhaar.uidai.gov.in/order-aadhaar-card](https://myaadhaar.uidai.gov.in/order-aadhaar-card)
- **Fee**: ₹50
- **Delivery**: 5-7 working days via Speed Post

### 6. Virtual ID (VID)

**Generate/Retrieve Virtual ID**
- **URL**: [https://myaadhaar.uidai.gov.in/genricVID](https://myaadhaar.uidai.gov.in/genricVID)
- **Purpose**: Create 16-digit temporary ID to use instead of Aadhaar number

### 7. Lock/Unlock Biometrics

**Biometric Lock Service**
- **URL**: [https://myaadhaar.uidai.gov.in/biometric-lock](https://myaadhaar.uidai.gov.in/biometric-lock)
- **Purpose**: Lock your biometrics when not in use for security

### 8. Aadhaar Linking

**Link Aadhaar with PAN**
- **URL**: [https://incometaxindiaefiling.gov.in/e-Filing/Services/LinkAadhaarHome.html](https://incometaxindiaefiling.gov.in/e-Filing/Services/LinkAadhaarHome.html)
- **Mandatory**: Required by Income Tax Department

**Link Mobile Number**
- Visit nearest Aadhaar center with Aadhaar and mobile
- SMS: Send "UID <Aadhaar number> <mobile>" to 51969

**Link Bank Account**
- Visit your bank branch with Aadhaar card
- Or use net banking to link online

### 9. Retrieve Aadhaar/EID

**Retrieve Enrollment ID**
- **URL**: [https://myaadhaar.uidai.gov.in/retrieve-eid-uid](https://myaadhaar.uidai.gov.in/retrieve-eid-uid)
- **Require**: Name, mobile/email used during enrollment

**Retrieve Aadhaar Number**
- **URL**: [https://myaadhaar.uidai.gov.in/retrieve-eid-uid](https://myaadhaar.uidai.gov.in/retrieve-eid-uid)
- **Options**: Using name and enrollment ID or name and biometrics

## Important Reminders

⚠️ **Beware of Fraud**
- UIDAI never asks for payment for enrollment or update
- Don't share Aadhaar details on unsecured websites
- Be cautious of fake Aadhaar-related websites

⚠️ **Use Official Portals Only**
- Always verify website URL starts with "uidai.gov.in"
- Download apps only from official app stores
- Don't trust third-party Aadhaar service websites
