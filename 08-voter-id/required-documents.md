# Required Documents for Voter ID Card

## Document Options

You need to provide documents to prove:
1. **Identity**
2. **Address** 
3. **Age/Date of Birth**

**Note**: One document can serve multiple purposes

## Acceptable Documents

### Proof of Identity, Address, and Age (Any ONE)

1. **Aadhaar Card** ✅ (Most preferred - covers all three)
2. **Passport**
3. **Driving License**
4. **PAN Card**
5. **Bank Passbook with photograph**
6. **Ration Card / PDS Photo Card**

### Additional Address Proof (Any ONE)

7. **Electricity Bill** (recent, not older than 3 months)
8. **Water Bill** (recent, not older than 3 months)
9. **Telephone Bill - Landline** (recent, not older than 3 months)
10. **Gas Connection Bill** (recent, not older than 3 months)
11. **Bank Passbook** (with address)
12. **Post Office Passbook**
13. **Rent Agreement** (registered/notarized)
14. **Property Tax Receipt**
15. **Credit Card Statement**

### Additional Age Proof (Any ONE)

16. **Birth Certificate** (issued by Municipal Corporation/Gram Panchayat)
17. **10th Class Certificate** (SSLC/Matriculation)
18. **Aadhaar Card** (if DOB mentioned)
19. **Passport**
20. **Kisan Card** (Passbook)
21. **Student ID Card** (from recognized institution)

### Additional Identity Proof (Any ONE)

22. **SSLC Certificate** (with photograph)
23. **Ration Card**
24. **Student ID Card** (recognized educational institution)

## Document Format

### Photographs
- Passport size (3.5 cm x 2.5 cm)
- Recent (not older than 3 months)
- Colored preferred
- Plain white/light background
- 2-3 copies

### Copies and Originals
- Self-attested photocopies
- Originals for verification
- Clear and legible copies

## Special Cases

### For First-Time Voters
- Birth certificate or 10th certificate (age proof mandatory)
- Address proof
- Identity proof
- Photograph

### For Address Change
- New address proof (any from list)
- Previous voter ID (if available)

### For Name Correction
- Document showing correct name
- Affidavit (if name change due to marriage/other reason)
- Previous voter ID

### For Duplicate Card
- FIR (if lost)
- Affidavit
- Identity and address proof
- Photographs

## Aadhaar Integration

**Aadhaar-based enrollment** (simplified process):
- Only Aadhaar number required
- Details auto-filled from Aadhaar database
- Faster processing
- No additional documents needed

## Document Checklist

Before applying:
- [ ] Identity proof (original + photocopy)
- [ ] Address proof (original + photocopy)
- [ ] Age proof (original + photocopy)
- [ ] 2-3 passport-size photographs
- [ ] Form 6 (filled and signed)
- [ ] Self-attestation on all copies

## Tips

✅ Use Aadhaar for quickest processing
✅ Ensure name consistency across documents
✅ Verify address matches with residence
✅ Keep extra photographs
✅ Self-attest all photocopies
✅ Carry originals for verification
