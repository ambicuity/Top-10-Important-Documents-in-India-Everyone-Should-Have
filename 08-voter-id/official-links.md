# Official Links and Resources for Voter ID

## Main Portal

### National Voters' Service Portal (NVSP)
- **Website**: [https://www.nvsp.in](https://www.nvsp.in)
- **Services**:
  - New voter registration (Form 6)
  - Search name in electoral roll
  - Apply for corrections (Form 8)
  - Apply for deletion (Form 7)
  - Transposition within constituency (Form 8A)
  - Download e-EPIC
  - Track application status

## Election Commission of India

### Official Website
- **URL**: [https://eci.gov.in](https://eci.gov.in)
- **Purpose**: Main ECI portal for election information

### Voter Services
- **URL**: [https://voters.eci.gov.in](https://voters.eci.gov.in)
- **Services**: All voter-related services and information

## Mobile Applications

### Voter Helpline App
- **Android**: [Google Play Store](https://play.google.com/store/apps/details?id=com.eci.citizen)
- **iOS**: [App Store](https://apps.apple.com/in/app/voter-helpline/id1440353105)
- **Features**:
  - Apply for new voter ID
  - Search electoral roll
  - Reportproblems
  - Track application
  - Find polling booth
  - BLO (Booth Level Officer) contact

## Services Available

### 1. Search Name in Electoral Roll
- **URL**: [https://electoralsearch.in](https://electoralsearch.in)
- **Purpose**: Verify if your name is registered

### 2. Download e-EPIC
- **URL**: [https://www.nvsp.in](https://www.nvsp.in)
- **Navigate to**: "Download e-EPIC"
- **Required**: EPIC number or reference number

### 3. Track Application Status
- **URL**: [https://www.nvsp.in](https://www.nvsp.in)
- **Navigate to**: "Track Application Status"
- **Required**: Reference number

### 4. Find Polling Booth
- **URL**: [https://www.nvsp.in](https://www.nvsp.in)
- **Navigate to**: "Know Your Polling Station"
- **Required**: EPIC number or search by details

## Contact Information

### National Helpline
- **Number**: 1950 (Toll-free, 24×7)
- **Purpose**: Voter registration queries, complaints

### Email Support
- **Email**: complaints@eci.gov.in (for complaints)
- State CEO emails vary

## Forms Download

### Important Forms
- **Form 6**: New voter registration
- **Form 7**: Objection for deletion of name
- **Form 8**: Correction of entries
- **Form 8A**: Transposition within constituency
- **Form 002**: Duplicate EPIC

**Download**: Available on NVSP portal and state CEO websites

## Grievance Redressal

### Online Complaint Portal
- **URL**: [https://www.nvsp.in](https://www.nvsp.in)
- **Navigate to**: "Grievance" section
- **Purpose**: Register complaints about voter services
