# How to Apply for Voter ID Card

## Online Application (Recommended)

### Through National Voters' Service Portal (NVSP)

**Website**: [https://www.nvsp.in](https://www.nvsp.in)

### Step-by-Step Process:

**Step 1**: Visit NVSP Portal

**Step 2**: Click on "Form 6" (for new voters)

**Step 3**: Fill Online Form
- Personal details (name, DOB, gender)
- Address details
- Mobile number and email
- Constituency details
- Upload photograph
- Upload documents

**Step 4**: Select Documents Type
- Choose from list of acceptable documents
- Upload clear scanned copies

**Step 5**: Submit Application
- Review all details
- Submit form
- Note reference number

**Step 6**: Print Application Form
- Download filled form
- Take printout
- Sign the form

**Step 7**: Submit at ERO Office (Optional)
- Can submit online documents only
- Or submit physical copy at Electoral Registration Officer office

**Step 8**: Field Verification
- Official may visit for verification
- Be available at given address

**Step 9**: Track Status
- Use reference number
- Check on NVSP portal

**Step 10**: Receive Voter ID
- By post to registered address
- Usually within 30-45 days
- Can download e-EPIC from portal

**Fee**: **FREE** - No charges

## Offline Application

### Step 1: Get Form 6
- Download from Election Commission website
- Or collect from Electoral Registration Officer office
- Or from local booth level officer

### Step 2: Fill Form 6
- Fill in CAPITAL LETTERS
- Provide all mandatory details
- Attach documents (self-attested)
- Attach passport-size photograph

### Step 3: Submit Form
- At Electoral Registration Officer (ERO) office
- Or give to Booth Level Officer (BLO)
- Or drop at designated centers

### Step 4: Get Acknowledgment
- Collect receipt with reference number

### Step 5: Verification
- Official will verify details
- May visit your address

### Step 6: Receive Voter ID
- By post within 30-45 days

## For Different Situations

### For First-Time Voters (Age 18)

**When to Apply**:
- As soon as you turn 18
- Special drives for 17+ year olds (enrolled after turning 18)

**Documents Required**:
- Age proof (10th certificate/birth certificate)
- Address proof
- Photograph

### For Person Shifting Address

**Within Same Constituency**:
- Fill Form 8 (for transposition)

**To Different Constituency**:
- Fill Form 6 in new constituency
- Fill Form 7 (deletion) in old constituency

### For Name/Details Correction

**Fill Form 8**:
- For correction in existing voter ID
- Provide correct details
- Submit proof documents

### For Lost/Damaged Voter ID

**Fill Form 002**:
- Apply for duplicate card
- Mention reason (lost/damaged)
- May need FIR (for lost card)

## Processing Time

- **Application to Verification**: 15-20 days
- **Verification to Issuance**: 10-15 days
- **Total Time**: 30-45 days

## Track Application Status

**Online**:
- Visit [https://www.nvsp.in](https://www.nvsp.in)
- Click "Track Application Status"
- Enter reference number

**SMS**:
- Send SMS to designated number (varies by state)

**App**:
- Use Voter Helpline app
- Check application status section

## Download e-EPIC

**e-EPIC** (Electronic Electoral Photo Identity Card):
- Digital version of voter ID
- Legally valid
- Can be downloaded from NVSP portal
- Available after voter ID generation

**How to Download**:
1. Visit [https://www.nvsp.in](https://www.nvsp.in)
2. Click on "Download e-EPIC"
3. Enter EPIC number or Form reference number
4. Verify with OTP
5. Download PDF

## Common Issues and Solutions

### Issue 1: Name not in electoral roll
**Solution**: Apply using Form 6, wait for processing

### Issue 2: Wrong details in voter ID
**Solution**: Apply for correction using Form 8

### Issue 3: Duplicate entry
**Solution**: Keep one, request deletion of duplicate using Form 7

### Issue 4: Voter ID not received
**Solution**: Track status online, contact ERO office

## Important Tips

✅ Apply well before elections (not during election period)
✅ Provide accurate information
✅ Use Aadhaar for quick processing
✅ Keep reference number safe
✅ Verify details before submission
✅ Download e-EPIC as backup
✅ Update mobile number for SMS updates

## Special Camps

Election Commission conducts special camps:
- Before elections
- At public places
- For on-the-spot registration
- Document verification done
- Bring all documents
